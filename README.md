## Instructions

When cloning the project, change the <code>sample.env</code> file name for <code>.env</code>. The project will run on **PORT 3000**.
```
REACT_APP_API_URL=http://localhost:8000/api/v1

```

Then, run:
```bash
npm install
```

To start the project run:
```bash
npm run start
```
